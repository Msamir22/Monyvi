# Logo Source Manifest: Egyptian Institution Presets

**Verified date**: 2026-05-25 **Scope**: Selectable providers from
`packages/logic/src/parsers/egyptian-bank-registry.ts`. **Asset policy**: Final
bundled assets must come from official provider-controlled sources only. Use
official brand/media kit first, official website logo asset second, and official
app-store icon only for wallet-specific cases. Do not use CBE, NTRA, directory,
news, Wikipedia, or other third-party assets as final bundled sources.
InstaPay/IPN is intentionally out of scope.

Local provider-specific logo files are bundled for every selectable known bank
and wallet. Manual/Other bank and wallet accounts continue to use the default
local fallback assets. Runtime UI must load only these bundled local assets and
must never fetch logo URLs remotely.

Some providers did not expose a clean direct website logo file. In those cases,
the bundled asset uses the official Google Play app icon when the listing is
published by the provider, or a provider-controlled regional/parent-site asset
when the Egypt site is unavailable. These cases are recorded below so they can
be replaced later if a cleaner provider media-kit asset becomes available.
